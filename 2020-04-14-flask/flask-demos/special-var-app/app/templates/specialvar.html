Affichage des variables spéciales

Cet exemple montre comment accéder aux différentes variables spéciales disponibles dans un template Jinja.

N'importe quelle URL peut être saisie comme par exemple:
/acajou/15/Demo?g=1&a=12

Variable g

{{ g }}
    {% for item in g %}
  • {{ item }} = {{ g[item] }}
    • {% endfor %}

Variable request

Objet request : {{ request }}

request caractéristiques

  • request.scheme = {{ request.scheme }}
  • request.url = {{ request.url }}
  • request.url_root = {{ request.url_root }}
  • request.path = {{ request.path }}
  • request.query_string = {{ request.query_string }}
  • request.user_agent = {{ request.user_agent }}
{# Extract the Arguments like:} request.args.get('a') #}

request cookies

{% if request.cookies %}
    {% for item in request.cookies %}
  • {{ item }} = {{ request.cookies[item] }}
    • {% endfor %}
{% endif %}

request args

{% if request.args %}
    {% for item in request.args %}
  • {{ item }} = {{ request.args[item] }}
    • {% endfor %}
{% endif %}

variable session

{{ session|default('session is not defined!') }}

variable Environment

{{ Environment|default('Environment is not defined!') }}

variable config

    {% for item in config %}
  • {{ item }} = {{ request.args[item] }}
    • {% endfor %}
